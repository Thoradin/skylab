Letöltési hibakódok

A letöltő szerver használata közben a letöltő kliens hibakódokkal jelzi a különböző problémákat. Amennyiben a rendszer által jelzett hibakód magyarázata nem segít a probléma megoldásában, javasoljuk, hogy ellenőrizze, hogy az operációs rendszer összes frissítése fel van-e telepítve vagy frissítse rendszerét a Microsoft Update oldalról.

FONTOS! A sikeres letöltéshez a letöltendő telepítőcsomag legalább kétszeresének megfelelő szabad hely szükséges a diszken!

Az egyes hibakódok magyarázata a következő:

  • SDC 00100
  • SDC 00200
  • SDC 00300
  • SDC 00400
  • SDC 00500
  • Error 70072
  • Error 70077

SDC 00100 Error

Unable to find the source CD (expecting to find a SDC file on a local CD).

SDC 00200 Error

A general error that is typically caused by:

  1. insufficient disk space to unpack or
  2. a disk error

SDC 00300 Error

Unable to read from media, which is typically caused by the PC being:

  1. unable to connect to network resource (for a Network Based install) or
  2. unable to read from CD (for a secure CD based install)

Please note that this is a network connection error only.

SDC 00400 Error

Unable to complete the download, which is typically caused by:

  1. an error in the configuration of the download package, usually due to a bad hosting location
  2. no Internet connection found
  3. Internet connection lost
  4. a firewall or proxy server between the client station and hosting location, or
  5. insufficient disk space to download

SDC 00500 Error

CRC check failed, which can mean that the package is corrupt and unusable. This is typically caused by:

  1. an incomplete transfer
  2. a corruption during transfer or
  3. a corrupt package on the host server

Error 70072

Install limit has been reached, which is typically caused by:

  1. the customer having attempted to download a product too many times to different folders (it is possible that no downloads have been successful)
  2. the customer having already successfully downloaded/installed the software, and now wanting to download again

Error 70077

Parameter authorization failure, which typically means that:

  1. no hosting location has been provided on the system for the product or
  2. the delivery client is corrupt (delete and download again).
